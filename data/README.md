# Sample Data

This folder describes the sample data the TechMart pipeline would process.
No live data files are included - this scaffold is for showing project
structure, not for running the pipeline.

## Files (described, not included)

- `orders.csv` - TechMart order transactions
- `customers.csv` - customer records

## Data Quality Issues

A real version of these files would contain typical problems for the
pipeline to handle:

- Duplicate orders
- Missing values
- Inconsistent date formats
